1. 基本概念

数据类型

  • 空 - None
  • 布尔 - True, False
  • 数 - 整数、长整数、浮点数、复数
  • 字符串 - '、"、'''、"""
    r"自然字符串，不转义"
    u"Unicode字符串"
  • list - 可变序列
    shoplist = ['apple', 'mango', 'banana']
  • tuple - 不可变序列
    zoo = ('wolf', 'tiger', 'monkey')
  • dict - 键值对集
    d = {key1: value1, key2: value2}

变量

var = 5 不用声明

缩进

缩进标识代码块，不能随便缩进

4. 函数

定义

global - 函数内声明全局变量

参数 & 默认参数 & 关键参数

默认参数要位于形参表末尾

可变参数 & 关键字参数

  • 可变参数 - *args，args接收的是一个tuple

return

  • 没有返回值的return 相当于 return None

  • 使用funcname.doc调用
